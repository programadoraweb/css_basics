Transitions

  1. transition: opacity 300ms ease-in-out 3s;
  2. transitions: [property - name of the property] [duration - how long it will happen ] [timing-function - ie. ease] [delay - how long should we wait till it starts]
  3. transition-property: color;
  4. transition-duration: 0.3s;
  5. transition-timing-function: linear;
  6. transition-delay: 1s;

Transition is the way how you get from point A to point B in a smooth maner
