Animation Structure

animation:
[name -of the anymation]
[duration - how long it will last]
[timing-function - ie. ease-in]
[delay - how long should it wait before the animation starts]
[iteraction-count - how many times the animation will repeat]
[direction - ie. normal, reverse, alternate]
[fill mode - where should the animaiton end]
[play state - ie. paused, running]

Keyframe structure:

@keyframes animationName { from { style } to { style } }

@keyframes animationName { 0% { style } 50% { style } 75% { style } 100% { style } }

The percentage allows you to make breaks on the movement
