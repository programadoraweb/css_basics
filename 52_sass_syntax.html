Sass Syntax

  1. Nesting Selectors is allowed

    2. Sass

    nav ul margin: 0 padding: 0 list-style: none li display: inline-block a display: block padding: 6px 12px text-decoration: none

    css

    nav ul { margin: 0; padding: 0; list-style: none; } nav li { display: inline-block; } nav a { display: block; padding: 6px 12px; text-decoration: none; }

  2. &__ or &-- or & , replaces the Parent Selector

    3. sass syntax

    .accordion { background: #f4f4f4 &__copy background: #646464

    Css Syntax

    .accordion { background: #f4f4f4; } .accordion__copy { background: #646464 }

  3. @Mixins & @include (resusable css)

    4. @mixin is used to set the style & @include to call/apply that style to an element

    sass

    @mixin button { display: inline-block; text-decoration: none; padding: 5em 7em; background-color: azure; &:hover { background-color: brown; } } .button { @include button; }

    css

    .button { display: inline-block; text-decoration: none; padding: 5em 7em; background-color: azure; } .button:hover { background-color: brown; }

  4. @for loop (for to x for through)

    5. sass

    @for $i from 1 through 3 ul:nth-child(3n + #{$i}) background-color: lighten($base-color, $i * 5%)

    css

    ul:nth-child(3n + 1) { background-color: #004080; } ul:nth-child(3n + 2) { background-color: #004d99; } ul:nth-child(3n + 3) { background-color: #0059b3; }

  5. @if / @else if & @ else

    6. You can include @if / @else if /@else statements inside your mixin declaration

  7. @each loop (its like mapping)

    8. sass

    $sizes: 40px, 50px, 80px @each $size in $sizes .icon-#{$size} font-size: $size height: $size width: $size

    scss

    .icon-40px { font-size: 40px; height: 40px; width: 40px; } .icon-50px { font-size: 50px; height: 50px; width: 50px; } .icon-80px { font-size: 80px; height: 80px; width: 80px; }

  8. @extend (reuse an existing class from another element)

    9. .btn{ background-color: red; } .p{ estend .btn; color: black; }
