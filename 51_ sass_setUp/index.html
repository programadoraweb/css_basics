VS Code Set Up (easiest option)

  1. enable Live Sass Compiler extension on Vs Code
  2. create a SCSS folder
  3. create style.scss or main.scss file
  4. click on Watch Sass on the bottom bar to generate a style.css or main.css file

    5. if the watch sass does not appers, update your extension

    if the style.css/ main.css file is generated inside the same folder (scss) as the .scss files, you can change the configuration on VS Code Settings + Extensions + Live Sass Compile Configuration + Live Sass Compile Settings: Formtas + Edit in settings.json & change "savePath": from "null" to "/css". So nexta time you save a file the .css will be generated inside the css folder.

  5. You will write your code on main.scss and it will appear on css folder automatically

Command Line Set Up (npm option)

  1. check you have npm installed
  2. npm install -g node-sass
  3. to compile: node-sass -w scss -o css
    4. (inside the scss file you write on the bash)

Document Structure

  1. SCSS folder
  2. COMPONENTS folder inside scss
  3. _fileName.scss with pieces of code eg. _colors.scss; _mixins.css or abstracts.css
  4. style.scss or main.scss to @import "_filename.scss"
