this is a div. You can convert a block element into an inline element

With Display-Inline-Block you can add width and height, whereas with Display- Inline it will always take by default the minimum width.

height, margin and padding have no impact on inline elements

IF you want to style an inline element eg. span, you need to make it a display-inline-block.
